Nossos Projetos

Eduardo: Cartão de Skills

Este aplicativo foi desenvolvido utilizando o framework Expo (React Native) com o objetivo de criar uma interface de cartão de visita digital personalizada.


Componentes utilizados:.


SafeAreaView, View, Text, Imagem, Card (React Native Paper).

Eduardo: Card de RPG

Este projeto consistiu no desenvolvimento de uma interface de jogo de RPG (Role-Playing Game) utilizando Expo React Native. Foram usados dois layouts de carta um vertical e outro horizontal


Componentes utilizados:


TouchableOpacity, Image (via URI), SafeAreaView, View (Containers Específicos)+.

Eduardo: Filmes

Este aplicativo foi desenvolvido para funcionar como um guia interativo de filmes. O foco principal do projeto foi a implementação de um sistema de navegação customizado e o uso de estados para alternar entre a galeria de filmes e a tela de detalhes individuais.


Componentes utilizados:


StyleSheet, Text, View, TouchableOpacity, Image, ScrollView.

Eduardo: Calculadora

Componentes utilizados:


TextInput, TouchableOpacity, ViewText, Stylesheet
